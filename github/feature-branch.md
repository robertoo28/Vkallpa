## Feature bracnhes

1. For each user story, you must create a feature branch, the name should be the same as the Epic name, and the user story name. 
2. When you finish all the development of the feature, you must do a PR to Main branch (Look at pull-request.md)
3. The name of the branch need to start with feature/  dont use codex in the name of the branch. 
4. Always commit your changes, to accept that a feature is finished, you must have the PR with the indications previusly mentioned.
